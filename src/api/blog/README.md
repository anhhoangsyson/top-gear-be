# Blog — README (tóm tắt)

Mục đích: Quản lý bài viết (blog) — nội dung, category, tag, comment.

Vị trí:
- `controller/`, `service/`, `repository/`, `schema/`, `dto/`.

Endpoints tiêu biểu:
- `GET /api/v1/blogs` — danh sách bài viết.
- `GET /api/v1/blogs/:slug` — chi tiết bài viết.

Ghi chú:
- Hỗ trợ SEO: slug, meta title/description.
