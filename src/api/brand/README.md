# Brand — README (tóm tắt)

Mục đích: Quản lý thương hiệu (brand) cho sản phẩm.

Vị trí:
- `controller/`, `service/`, `repository/`, `schema/`.

Endpoints:
- `GET /api/v1/brands` — lấy danh sách thương hiệu.
- `POST /api/v1/brands` — tạo brand (admin).

Ghi chú:
- Dùng để lọc sản phẩm theo brand và hiển thị thông tin thương hiệu.
