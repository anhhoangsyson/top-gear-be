# Laptop — README (tóm tắt)

Mục đích: Module chuyên cho sản phẩm dòng laptop (nếu dự án có custom logic cho laptop).

Vị trí:
- `controller/`, `service/`, `repository/`.

Usage:
- Các endpoint/logic riêng cho laptop (filter theo cấu hình, so sánh cấu hình).

Ghi chú:
- Nếu nhiều dòng sản phẩm có logic tương tự, cân nhắc gom chung vào `product` và dùng type/brand.
