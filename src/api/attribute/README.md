# Attribute — README (tóm tắt)

Mục đích: Quản lý thuộc tính sản phẩm (ví dụ: RAM, CPU, màu sắc).

Vị trí:
- `controller/`, `service/`, `repository/`, `schema/`.

Usage:
- Các attribute dùng để hiển thị filter trên frontend và tạo variant.

Ghi chú:
- Thường là static data; có thể cache để giảm truy vấn DB.
