# Category — README (tóm tắt)

Mục đích: Quản lý danh mục sản phẩm (categories / menus).

Vị trí:
- `controller/`, `service/`, `repository/`, `schema/`.

Endpoints tiêu biểu:
- `GET /api/v1/categories` — lấy danh mục (cây/hierarchical nếu có).
- `POST /api/v1/categories` — tạo category (admin).

Ghi chú:
- Danh mục thường dùng để phân filter, SEO và hiển thị menu.
